This repository contains the code for the CLARE-X project. It is designed for **CLEAR-X: Contrastive Learning with Explainable Re-ranking for Academic Reviewer Recommendation**.

## Prerequisites

Before running the code, ensure you have installed all required dependencies. You can install them by running:

```bash
pip install -r requirements.txt
```
## Usage
To run the project, use the following command:
```
python main.py --config nips.yaml --epochs 200
```
